Kasumi Lighting Model

Introduction

Kasumi Lighting Model is stylized render pipeline for Unity I made in my free time written in HLSL, Shaderlab and C#. It is designed to be generic, meaning that it is a "meta" pipleine and is not limited to a specific artistic style. Artists can achieve various styles by using different parameters the pipeline provides: (The pictures below demostrate some of the styles I created)

Pipeline Architecture

Core Features

Stylized Lighting Model
  • Custom Ray Diffusion Model and Color Model
  • Stylized Metallics, smoothness, Clear Coat and BRDF
  • Ambient Occulusion and Global Illumination
Traditional Workflow Support
  • Alpha map, normal map, specular map...
  • Transparency and Alpha cutout, culling
  • Layer Blending
  • Vertex Animation
Stylized Effects
  • Custom Object Outline
  • Custom Shadow
  • Autofill surface with grass/snow/sand..
  • Color Interpolation
Additional Components
  • Stylized Water
  • Depth Fog
  • Bidirectional Gaussian Blur Post Processing
