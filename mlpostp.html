Unity Neural Network Rendering

Introduction

I'm perhaps among the first group of technical artist in the world (!) that experiments on Neural Network Rendering in Unity, using the internal beta package Sentis (Previously known as Barracuda). It is designed to be the neural network interface for Unity.
In the work I added a Tensorflow VAE neural network to the unity's default render pipeline, so that I could achieve arbitary style transfer in real time given a style reference image.
